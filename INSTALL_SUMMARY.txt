🚀 QUICK START FOR VPS:
-----------------------
1. Extract: unzip rimarus-fixed.zip
2. Setup: cd rimarus && python3 -m venv venv && source venv/bin/activate
3. Install: pip install -r requirements.txt
4. Configure: cp .env.example .env && nano .env
5. Run: cd Zohun && python __main__.py

📋 REQUIRED CREDENTIALS:
------------------------
Before running, you MUST configure in .env:
- API_ID and API_HASH from https://my.telegram.org
- BOT_TOKEN from @BotFather
- OWNER_ID (your Telegram user ID)
- LOG_SELLER and LOG_BACKUP (channel IDs)
